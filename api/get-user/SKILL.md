# get-notion-user

use a python script to perform HTTP GET requests to the notion API. the script automatically resolves authorization tokens from the .env file.

## commands

```powershell
python api/get-user/scripts/get-notion-user.py
```

## guidelines

- use this to retrieve information about the currently authenticated bot user.
